# Anishka Autofill Chrome Extension

This is a separate Chrome extension folder that leaves the existing `chrome-extension` directory untouched.

It autofills common fields with:

- Name: `Anishka Jannu`
- Email: `anishkajannu@gmail.com`
- Phone: `5104585877`
- College: `Carnegie Mellon University`

## Load it in Chrome

1. Open `chrome://extensions`
2. Turn on Developer mode
3. Click **Load unpacked**
4. Select this folder:

`anishka-autofill-extension`

## What it fills

- Full name
- First name
- Last name
- Email
- Phone
- College / university / school / institution

## How to use it

1. Open any form page in Chrome
2. Click the extension icon
3. Click **Autofill current page**
4. If you want to preview first, click **Scan fields**
